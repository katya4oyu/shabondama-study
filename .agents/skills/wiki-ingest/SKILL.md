# Wiki Ingest

Use this skill to integrate knowledge into `wiki/`.

1. Read `wiki/README.md`, `wiki/index.md`, and `wiki/log.md`.
2. Search before writing with `rg`.
3. Prefer updating existing pages over creating new pages.
4. Use required frontmatter for every page.
5. Connect pages with regular Markdown links. Do not use `[[WikiLinks]]`.
6. Append a short `ingest` or `maintenance` entry to `wiki/log.md`.
