# Wiki Query

Use this skill to consult `wiki/` before answering.

1. Read `wiki/index.md` to choose a likely route.
2. Search exact terms with `rg`.
3. Read the most relevant wiki pages and linked durable docs or issues.
4. State whether the wiki was useful, incomplete, stale, or misleading.
5. If the query reveals missing knowledge, update the wiki or append a short `query` entry to `wiki/log.md`.
