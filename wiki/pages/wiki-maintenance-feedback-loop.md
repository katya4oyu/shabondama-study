# Wiki Maintenance Feedback Loop

When a wiki page helps, make that route easier to find. When it fails, leave a
small repair behind.

## Loop

1. Start from the index when the task shape is known.
2. Search exact terms with `rg`.
3. Read the smallest useful set of pages.
4. Record whether the page helped.
5. Fix stale text, missing links, poor routes, or misleading claims.
6. Append a short entry to the wiki log.
